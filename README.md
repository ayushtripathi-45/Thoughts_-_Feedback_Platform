# Thoughts & Feedback Platform

A full-stack web application to create, manage, and delete personal thoughts and feedback.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL
- Authentication: JWT

## Features
- User authentication (Login / Signup)
- Create & delete thoughts
- Protected dashboard
- JWT-based authorization

## Setup
1. Clone the repository
2. Install dependencies  
   `npm install`
3. Create `.env` 
4. Start server  
   `node server.js`

## Note
This project is for learning and practice purposes.
